# 3D打印摆摊助手

一个用于3D打印商品摆摊的管理系统，支持商品管理、销售统计、成本计算等功能。

## 技术栈 其他

- **前端**：React + Vite
- **后端**：Node.js + Express
- **数据库**：SQLite
- **图片处理**：sharp
- **认证**：bcrypt + JWT

## 功能特性

### 前台功能
- 商品列表展示
- 商品搜索
- 今日推荐和今日特价
- 管理员入口

### 后台功能
- 商品管理（添加、编辑、删除）
- 图片自动压缩
- 耗材使用情况管理
- 成本自动计算
- 销售记录和统计
- 数据备份与还原
- 设置管理
